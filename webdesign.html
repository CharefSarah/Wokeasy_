UI/UX

UX: Il se concentre sur l'organisation de l'interface, en vérifiant que les transitions sont utiles, que les pages sont correctement liées et offrent les fonctionnalités attendues par les utilisateurs. Pour ce faire, les concepteurs UX se mettent à la place de l'utilisateur moyen afin de comprendre comment les gens interagissent avec leurs applications à l'échelle mondiale. Les prototypes et les wireframes (par exemple Adobe XD, Figma, Sketch, etc.) ainsi que les cartes mentales sont des outils utiles pour sa tâche.


UI: Interface utilisateur, le mot parle de lui-même. Par conséquent, les concepteurs d'interface utilisateur se concentreront sur l'apparence d'une interface de site Web ou sur la présentation graphique d'une application.
